**Parkour-Ruokrap** adalah sebuah game parkour 3D sederhana yang dikembangkan menggunakan **Unity** sebagai bagian dari Proyek Akhir mata kuliah **Grafika Komputer**. Game ini mengusung konsep **endless runner** dengan gaya visual **voxel** yang terinspirasi dari Minecraft.

Pemain akan mengendalikan karakter voxel yang terus berlari di atas lintasan yang di-generate secara **procedural** tanpa batas. Tidak ada kondisi menang dalam game ini â€” tujuan utama pemain adalah **bertahan selama mungkin** dengan menghindari jurang yang muncul secara acak.

- **Genre**: Endless Runner / Parkour  
- **Engine**: Unity

Lintasan atau ground akan terus dibuat secara otomatis menggunakan sistem **procedural generation**, sehingga setiap sesi permainan memiliki pola yang berbeda. Tantangan utama pemain adalah menjaga keseimbangan, mengatur timing lompatan, dan bereaksi cepat terhadap perubahan lintasan.

Jika pemain terjatuh ke dalam jurang, permainan akan **restart secara otomatis** dari awal.

## âœ¨ Fitur Utama

- **Procedural Ground Generation**  
  Ground terus di-generate secara otomatis tanpa batas.

- **Endless Runner System**  
  Permainan berjalan terus hingga player gagal bertahan.

- **Double Jump dengan Rotasi**  
  Player dapat melakukan double jump. Saat double jump, karakter akan berputar untuk memberikan efek visual yang lebih dinamis.

- **Power-Up**  
  Power-up muncul secara acak. Ketika diambil, ukuran (scale) player akan membesar sementara waktu.

- **Auto Restart**  
  Game akan langsung dimulai ulang ketika player jatuh ke jurang.
